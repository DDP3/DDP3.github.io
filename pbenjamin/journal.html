Here are today's main ideas:

Today we started JavaScript. Spacing is important in JavaScript mainly to keep it organized and easy to read, like any other code, but occasionally it makes a difference, like needing a space after a var tag.

A variable is a little box that is used to store information, such as letters or numbers. (similar to Scratch variables)

Strings are words or text inside either double or single quotes. A string must end with whatever quote type it started with. So if a sting starts with a double quote, it must end with a double quote. If both types of quotes are used in a string, you can put a \ before that quote.

Numbers are used for calculations, keeping track of scores, or anything that is being counted in a variable.
