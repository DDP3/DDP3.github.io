Tuesday, February 17th, 2015

Today, the topics of study were variables, strings, spacing, and numbers in the context of JavaScript.

  • Variables are ways to store information in computer code. A variable can be assigned data with a specific value type to be used later. They are defined by placing var, then the variable name, such as varName, and then = value. Values come in many different types.
  • One type of value is a string. Strings are groups of characters surrounded by quotation marks or single quotes. They are read by the interpreter as characters with no special meaning. To place quotes inside a string without breaking it, an escape character, like a backslash, must immediately precede it.
  • Numbers are another type of value. Unlike strings, they are interpreted to have a specific meaning. They are used for computations in JavaScript.
  • Spacing is not completely necessary in JavaScript, but it should be added to ease readability. In naming, spaces are crucial, such as separating the var from the variable name, and having no spaces in the variable name. Other than that, even line breaks are acceptable!

Today, we learned how to assemble an array, which is a variable that can hold multiple values, and how to call each of those individual values using call numbers. Arrays hold each value between brackets, and the strings, functions, booleans, integers and other data types are separated by commas. Calling a specific value requires calling the variable, immediately followed by two brackets with a number corresponding to its position, with the first value having zero as that number.
