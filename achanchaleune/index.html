Autobiography

Hello. I, Anthony Chanchaleune, designed this site the way I did because I wanted to satisfy the requirements in order to pass my project. I also designed it so that it would look good on mobile devices and desktop computers.

The rough outline of this site began with the online journal entry. My journal started out pretty basic using just normal HTML and CSS styling. Then I did the Rock, Paper, Scissors project. Again, this was also pretty simple in terms of layout and design. But making these websites allowed me to practice building a foundation and basis for web designing.

After being introduced to Bootstrap, the evolution of this website really began to get going, at least in terms of style. It allowed for things such as an organized grid layout that will be used extensively throughout the website. It also allows for stylistic push and pull that will make this website look better for both mobile and desktop versions. Bootstrap has has aided in making a more modern, advanced, and prettier site.
